# Sistema de Barberia

Sistema de consola hecho con Python para gestionar citas de una barberia

## Funciones actuales

- Registrar citas
- Ver citas
- Marcar citas como atendidas
- Guardar datos en JSON
- Generar ID automaticamente
- Evitar IDs repetidos
- Guardar cambios al marcar citas como atendidas
